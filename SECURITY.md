# Security Policy

## The Immune System

YuánGūnGūn OS has a built-in immune system for AI — multi-layer defense against prompt injection, adversarial inputs, and manipulation attacks.

But no system is perfect. We need your help to make it stronger.

---

## Reporting a Vulnerability

**Do NOT open a public issue for security vulnerabilities.**

Instead, email: **yuangungun-hack@coze.email**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (if any)

We will respond within 7 days and keep you updated on the fix.

---

## Scope

**In scope:**
- Prompt injection attacks
- Adversarial input crafting
- Model extraction / data leakage
- Memory manipulation
- Immune system bypass
- On-device data exposure

**Out of scope:**
- Social engineering (phishing the user)
- Physical device theft
- OS-level vulnerabilities (report to Google/Apple)

---

We're a small team building something ambitious. While we can't offer cash bounties yet,

- 🏆 Public acknowledgment in our security hall of fame
- 🐼 Direct collaboration with the ShadowEdge security team
- 🛡️ Early access to new immune system features
